# Calm Web

**Calm Web** is a minimal-mode web simulator that contrasts noisy “Chaos Mode” pages with clean, distraction-free “Calm Mode.”

 **Built with:** HTML · CSS · JavaScript (no frameworks)

---

##  Features
- Toggle between Chaos and Calm modes  
- Real-time reading-preference sliders (font size, line height, width)  
- Ambient Web Audio hum for focus  
- Distraction score counter  
- Bookmarklet generator to declutter any page  
- Persistent settings via `localStorage`
